# Netflix EDA Project

A simple exploratory data analysis project based on the **Netflix Titles dataset**.

## What's Included
- `netflix_eda.ipynb` — Jupyter Notebook with full analysis  
- `netflix_titles.csv` — Dataset  
- `Netflix_Styled_EDA_Presentation.pptx` — Presentation  
- `graphs/` — All generated charts  

## How to Run
1. Install required libraries:
   ```
   pip install pandas matplotlib numpy python-pptx
   ```
2. Open the notebook:
   ```
   jupyter notebook netflix_eda.ipynb
   ```

## Summary
This project explores:
- Movies vs TV Shows distribution  
- Release year trends  
- Ratings breakdown  
- Duration patterns
